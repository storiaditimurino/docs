# Export Private Key

You can export the private key for any wallet to back it up or use it in another application.

## How to Export

1. Go to **Wallet** \> tap the wallet you want to export
2. Tap **“Export”**
3. The bot sends a message containing your private key in hex format (`0x...`)

## Security

<Warning>
  Your private key gives full control over your wallet and funds. Never share it with anyone.
</Warning>

The private key message includes important security features:

- The message **auto-deletes after 60 seconds** for security
- You can tap **“OK”** to delete it immediately 
- The key is displayed in a `monospace code block` for easy copying\

## Best Practices

- Copy the key and store it in a secure location (password manager, encrypted note, or paper backup)
- Do not screenshot the key in shared or public spaces
- Do not send the key to anyone — PolyX support will never ask for it

## Created vs Imported Wallets

- **Created wallets** — you can export the seed phrase to recover all created wallets at once
- **Imported wallets** — each has its own private key that should be backed up separately
