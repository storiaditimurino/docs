# Settings

Customize the quick buy and sell preset amounts that appear on your trading screens. Access settings from the main menu or with`/settings`.

## Quick Buy Presets

You have **3 buy presets** — dollar amounts that appear as one-tap buttons when buying shares. Default values are shown on the buttons (e.g.,10,10,50, \$100).To change a buy preset:

1. Open **Settings** (main menu \> Settings, or `/settings`)
2. Tap the preset you want to change (e.g., “\$10”)
3. Type a new USD amount (e.g., `25` or `250`)
4. The preset is updated and the settings screen refreshes

Buy presets have a minimum amount (set by the platform). If you enter a value below the minimum, you’ll be prompted to enter a higher amount.

## Quick Sell Presets

You have **4 sell presets** — percentages of your position to sell with one tap. Default values might be 25%, 50%, 75%, and 100%. To change a sell preset:

1. Open **Settings**
2. Tap the percentage you want to change (e.g., “25%”)
3. Type a new integer between 1 and 100
4. The preset is updated

## Where Presets Appear

Your presets are used throughout the trading experience:

| **Preset Type** | **Where It Appears** |
| :-- | :-- |
| **Buy presets** | Market buy screen, Position Detail (Market tab buy section), Limit buy amount selection |
| **Sell presets** | Position Detail (Market tab sell section) |

## Switching Between Presets

While editing one preset, you can tap another preset button to switch to editing it instead — the bot cleans up the previous prompt and starts a new one.

## Cancelling

Tap **“Cancel”** during text input to discard your change and return to the settings screen with the original values.
