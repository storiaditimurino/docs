# AI Insights

PolyX can generate AI-powered analysis for any Polymarket event or market. Use the `/insight` command to get started.

## How to Use

1. Type `/insight` in the chat
2. The bot prompts you to paste a Polymarket URL
3. Paste an event or market link (e.g., `polymarket.com/event/...` or `polymarket.com/event/.../market-slug`)
4. Merlin displays the market/event and generates an AI analysis

## What You Get

AI Insights provide analysis of the market or event, helping you make more informed trading decisions. The analysis is generated based on the current market data and conditions.

## Credits

Insights use a credit system:

- Each insight costs **1 credit**
- If you’ve already unlocked insights for an event, viewing it again is free
- When you’re out of credits, you’ll see a message with options to get more

<Tip>
  If you paste a market URL that belongs to an event you’ve already unlocked, the insight is free.
</Tip>

## Alternative Access

Besides the `/insight` command, you can also trigger insights from the Event screen by tapping the **“Event Insights”** button (when available). Both paths use the same credit system and produce the same analysis.

## Supported URLs

The `/insight` command accepts:

- Event URLs: `polymarket.com/event/{slug}`
- Market URLs: `polymarket.com/event/{slug}/{market-slug}`

If you paste something that isn’t a valid Polymarket URL, the bot will ask you to try again.
